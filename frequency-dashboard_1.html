üì• How to Input Data

Enter your raw numerical data separated by commas.

Example: 100, 105, 110, 115, 120, 125

You can paste data from Excel or any spreadsheet.

Number of Classes: Usually 5-20. Start with 6 classes and adjust if needed.

The app will:

  1. Calculate the range of your data
  2. Determine class width automatically
  3. Create frequency distribution table
  4. Generate all four graphs
üìä Understanding the Table

Class Limits: The range labels (e.g., 100-109)

Class Boundaries: More precise with decimals (99.5-109.5) to prevent gaps

Midpoint: Center of the class (e.g., 104.5). Used for frequency polygon.

Frequency: How many data values fall in this class

Cumulative Frequency: Running total up to this class

Relative Frequency: Percentage (frequency √∑ total √ó 100%)
